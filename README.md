# Chat Application

A simple real-time chat application built using **HTML**, **CSS**, and **JavaScript**.  
It allows multiple users to send and receive messages instantly in a clean and responsive interface.

# Features
- Real-time messaging  
- Responsive and clean UI  
- User-friendly design  
- Lightweight and fast  

# Technologies Used
- HTML5– structure  
- CSS3 – styling and layout  
- JavaScript (ES6) – functionality
